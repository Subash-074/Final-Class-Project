Python Django Backend E_commerce Application with Admin, User Login, Logout Frontend View  Built using Python, Django, CSS

Steps to Run in Terminal:
Right Click on the project folder.
Open the terminal. 


Type: cd puddle, Then press enter, 

Terminal should look like:
PS C:\Users\subas\Documents\GitHub\Final-Class-Project\puddle>

Activate the virtual environment
env\Scripts\activate

You should Now See:
(env) PS C:\Users\subas\Documents\GitHub\Final-Class-Project\puddle>

Now run this 
python manage.py runserver


Then Copy Server link available on terminal  in chrome browser: 

This project usually works in this link after completing above mentioned steps:
http://127.0.0.1:8000
